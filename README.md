# Hangman Game in C

This is a console-based Hangman game written in C.
I built this project as a first semester BS Computer Science student
to practice programming fundamentals.

## Features
- Choose category (Celestial bodies/Counties/Programming Languages)
- Word list stored in `countries.txt`, `progLang.txt`, and `space.txt`
- Shows guessed letters and remaining attempts
- shows hangman figure
- Simple input validation

## Concepts Used
- Variables and data types
- Loops
- Conditional statements
- Functions
- Arrays and strings


## Purpose
This project helped me understand basic logic building and C programming.


## How to Run
Compile the program using any C compiler.
